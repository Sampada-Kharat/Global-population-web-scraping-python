# Global-population-web-scraping-python
Web scraping project using Python, Requests, and BeautifulSoup to extract country-wise world population data, clean it with pandas, and export it as a structured CSV file for analysis.

## World Population Web Scraping

This project scrapes country-wise world population data from a public website using Python. The data is extracted with Requests and BeautifulSoup, cleaned using pandas, and saved as a CSV file for further analysis or visualization.

## Tools & Libraries
- Python
- Requests
- BeautifulSoup
- pandas

## Workflow
1. Send HTTP request to the target webpage
2. Parse HTML using BeautifulSoup
3. Extract population table data
4. Clean and format the dataset
5. Export data to CSV

## Output
- Cleaned CSV file containing country-wise population data

## Use Cases
- Exploratory Data Analysis (EDA)
- Power BI / Tableau dashboards
- Population trend analysis
